Snappy Cash

Help Page

Settings

  • convert automatically: [Default enabled] This extension will attempt to find all plain-text Bitcoin Cash addresses on the page while adding options to the right. Address searching is persistent to dynamically loaded content. Disable if performance seems impacted.
  • convert selections: [Default disabled] When you select (highlight) an address, options will then be added to the right. Currently, only automatic or selection conversions can be enabled at once.
  • integrate Badger Wallet: [Default enabled] Badger Wallet is an extension developed by Bitcoin.com. With Badger Wallet installed on your browser as well as this extension, a widget that will appear next to the QR code option. This extension injects code to interact with Badger Wallet's "web4bch" API, which our access to is blocked by some sites; such as, Twitter.
  • ignore list: On a site you wish to disable this extension on, click the add button (+ icon) to ignore current site in the future. Remove button (- icon) will appear next to ignored sites to remove from list.
  • Page refreshes are required for all settings to take effect.

Please open issues in the GitHub repository that you encounter.
